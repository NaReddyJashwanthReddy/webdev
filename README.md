## Counter App

The **Counter App** is a simple web application that demonstrates interactive UI updates using HTML, CSS, and JavaScript.

### Features
- Increment and decrement the counter value with buttons
- Modern, clean UI with a blurred background image
- Responsive and visually appealing design

### How to Run
1. Open the `counter/index.html` file in your web browser.
2. Use the Increment and Decrement buttons to change the counter value.
